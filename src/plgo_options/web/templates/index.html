Load the current book, choose a target maturity, then run. Same engine as Optimizer v2.

Scope

The expiry the optimizer rolls/rebuilds the book into. Required.

Ctrl/Cmd-click for multiple. None selected = all counterparties.

Objective weights

Weight on matching the target payoff curve vs. trading/collateral cost. Higher = prioritize the fit even if trades cost more.

Penalty on falling short of target vs. exceeding it. 1 = symmetric; >1 treats the target as a floor (undershoot hurts more).

Blend fitting the target today (0) vs. 90 days forward (1). e.g. 0.3 = 70% today / 30% T+90, so theta drift is considered.

Holding-cost weight on posted collateral. Higher = work harder to avoid growing collateral usage; 0 = collateral is free.

Penalty on premium-paid vs. collected imbalance per counterparty. Higher = trades self-fund more. Soft cost, not a hard cap.

Constraints

Fraction of mark value assumed lost when closing a position (0.2 = 20% slippage on unwind).

Per-unit cost for opening brand-new positions, biasing the LP toward reusing / rolling existing ones.

Hard per-counterparty cap on gross collateral change (0 = no increase, 0.05 = +5%, -0.1 = force −10%). Clear to disable.

Hard cap on any single candidate's size (contracts) — stops the LP proposing unrealistically large single lines. Clear to disable.

Cap on Replacement/New line items shown (unwinds & box legs always kept). Top-N by notional after solving. Clear to disable.

Rolls

Roll positions with DTE ≤ this. Clear to disable forced rolls. −1 = force-roll exactly the trades checked in Trade Management.

When rolling, only touch in-the-money positions.

Transaction cost — VOL pts

One-way bid-ask half-spread in implied-vol points, per counterparty. The engine charges cost = |vega| × VOLpts on each executed leg. Loaded from the book; edit any value.

Load the risk profile to list counterparties.

Output

Add a per-counterparty box spread (flat vs. spot) sized to neutralize leftover cash imbalance. Uncheck to see the raw result.

              How to read this chart
              
    
                
  • Before (current book) is your whole book today; After is the book once the proposed trades are applied. The gap between them is the net effect of the trades (unwinds + new) — the "After" line is not the new trades on their own.
    • 
                
  • Both curves are P&L from today, anchored to $0 at the current spot, so they both cross zero at spot. The premium you pay/receive is not visible on the curve — read it from the Net Premium / Cash Shift tiles above.
    • 
                
  • Roll candidates — tick to unwind (below left) lists positions eligible to roll (DTE ≤ threshold), ITM & non-ITM, with ITM pre-ticked. Ticking sets exactly what the optimizer rolls on the next Run and updates the dashed Book excl. ticked unwinds line.
    • 
                
  • X-axis is log spot price (works for ETH and FIL). Hover anywhere for all values at that spot. Click a legend entry to hide a line; double-click to isolate it.
